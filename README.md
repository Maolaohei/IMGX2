**Image Pro Zoomer (Mix01 Edition)** 是一款专为**图片审核**和**高清预览**设计的专业级 Chrome 浏览器扩展。它能够自动识别各大主流平台的缩略图，并实时解析出对应的高清原图，提供如电子显微镜般的丝滑缩放体验。

## 🚀 核心特性

-   **智能高清解析引擎**：内置强大的规则库（已实现规则解耦，存储于 `rules.json`），支持 Amazon、京东、淘宝、天猫、Twitter、微博、Pixiv、Bilibili、蜜柑计划 (Mikan Project) 等数十个国内外主流平台的高清原图直出。
-   **三模式平滑切换**：
    -   🔍 **局部放大**：传统的放大镜模式，在小窗内查看细节。
    -   🖼️ **整体跟随**：大图预览，悬浮窗口紧随鼠标移动。
    -   🖥️ **居中模式**：全屏暗场遮罩，大图固定在屏幕中央审计。
-   **小型图片智能优化**：针对极小图（如 50px 或 100px 以下的缩略图）自动启用激进的 9 倍或 6 倍初始缩放，确保细节不被遗漏。
-   **画质状态提示**：放大镜右上角实时显示当前是「原图放大」还是「高清图解析放大」，可在设置中一键开关。
-   **突破视图限制**：支持开启“突破限制”模式。当图片分辨率远超屏幕时，外框固定，内部图像随鼠标位移实时平移（Panning），扫描每一个像素细节。
-   **右键高清保存**：提供专门的右键菜单，支持直接触发浏览器下载高清原图。文件名自动补全后缀，并严格遵循 `文件名_原图.格式` 的命名标准。
-   **极致性能优化**：核心渲染引擎基于 `requestAnimationFrame` 动画帧开发，完美解决高频滑动、相邻图片快速切换时的卡顿与漏图问题。

## 🛠️ 安装方法

1.  下载本项目的所有代码（`manifest.json`, `rules.json`, `content.js`, `background.js` 等）至同一个文件夹。
2.  打开 Chrome 浏览器，访问 `chrome://extensions/`。
3.  在右上角开启**“开发者模式”**。
4.  点击**“加载已解压的扩展程序”**，选择包含本项目代码的文件夹。
5.  点击浏览器右上角插件图标，阅读并**同意免责声明**后即可开始使用。

## ⌨️ 快捷键操作 (默认)

| 键位 | 功能说明 |
| :--- | :--- |
| `V` | 循环切换视图模式 (局部 -> 跟随 -> 居中) |
| `R` | 顺时针旋转图片 90° |
| `M` | 水平镜像翻转图片 |
| `+` / `=` | 增加放大倍数 (步进 0.5x) |
| `-` | 减少放大倍数 (最低 1.5x) |

## ⚙️ 规则配置 (`rules.json`)

规则与逻辑已实现完全分离。您可以直接修改 `rules.json` 文件来添加对新网站的支持，无需修改核心脚本：

```json
{
  "name": "示例平台",
  "hosts": "example\\.com",
  "match": "/thumb/(\\w+)\\.jpg",
  "replace": "/original/$1.png"
}
