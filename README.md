# 🌌IMGX2 浏览器媒体扩展

![Version](https://img.shields.io/badge/version-2026.04-blue.svg)
![Platform](https://img.shields.io/badge/platform-Chrome%20%7C%20Edge-green.svg)
![License](https://img.shields.io/badge/license-MIT-orange.svg)

专为 Twitter (X) 和 Pixiv 等高度动态化网站打造的**极客级浏览器媒体扩展**。
通过主世界注入、React Fiber 状态逆向、Canvas 帧克隆以及原生 GraphQL API 劫持，突破前端 UI 限制与反爬机制，为你提供无与伦比的沉浸式看图与视频下载体验。

---

## ✨ 核心黑科技 (Core Features)

### 🎥 终极视频解析与播放
* **React Fiber & GraphQL 逆向提取**：按下 `D` 键，瞬间打入网页主世界 (Main World)，读取推特底层的 `__reactFiber` 状态树，或伪装调用官方 `TweetResultByRestId` GraphQL API，100% 提取最高码率的 `.mp4` 直链。
* **反-反暂停机制**：突破推特基于 IntersectionObserver 的离开视野自动暂停限制。
* **自定义沉浸进度条**：彻底抛弃原生繁琐的 Video UI，屏幕底部提供极简流光进度条，支持点击精准快进/后退。

### 🌌 沉浸式无痕画廊 (Immersive Gallery)
* **全局遮罩与抗污染渲染**：采用脱离文档流的极高层级 DOM 渲染，免疫 React/Vue 站点的 CSS 样式污染（告别画面发暗、变形）。
* **智能加载状态机**：无底图时显示推特蓝居中 Loading 环；高清图解析时右上角触发“电池充电式”扫描流光特效。
* **自动翻页引擎**：使用左右方向键浏览时，到达底部会自动触发页面滚动并加载更多动态，真正实现“无限续杯”。

### 👻 幽灵交互系统 (Phantom Interaction)
在完全黑屏的沉浸模式下，无需退出即可完成社交操作：
* **无痕点赞 (`L`)**：自动寻找并触发当前媒体所属推文/作品的 Like 按钮。
* **隐秘关注 (`F`)**：绕过推特复杂的下拉菜单，一键完成 Follow/Unfollow 状态切换。
* HUD 面板实时反馈当前作者的关注状态与喜欢状态。

### 🧩 极致解耦架构 (Decoupled Architecture)
* **`content.js` (渲染主引擎)**：只负责快捷键监听、UI 绘制、多模式放大镜（局部、跟随、避让）、图片跨域打包。
* **`immersive-rules.js` (独立适配驱动)**：采用插件化设计，利用正则缓存池精准匹配域名。将所有脏活累活（寻找 DOM、过滤广告推文、读取关注状态、视频直链提取）全部隔离在此，确保主引擎的绝对稳定。

## ⌨️ 极客快捷键 (Keyboard Shortcuts)

在任意适配网页（如 X.com）触发：

| 快捷键 | 功能描述 | 备注 |
| :--- | :--- | :--- |
| `Ctrl` + `F12` | 🌌 **进入/退出沉浸模式** | 自动捕捉当前视野内最大的媒体 |
| `Q` | ⏯️ **播放 / 暂停视频** | 具备防误触和防系统劫持机制 |
| `D` | 📥 **提取并下载最高画质视频** | 优先调用原生按钮，失败则触发 GraphQL / Fiber 逆向 |
| `S` | 💾 **保存高清原图** | 突破防盗链，直接打包至 `IMG_Download` |
| `L` | ❤️ **喜欢 / 取消喜欢** | 底部 HUD 会实时提示状态 |
| `F` | 🫂 **关注 / 取消关注** | 同上 |
| `←` / `A` | ⬅️ **上一项** | 到顶时自动向上滚动加载 |
| `→` / `D` | ➡️ **下一项** | 到底时自动向下滚动加载 |
| `V` | 🔄 **切换视图模式** | 局部放大 / 整体跟随 / 智能避让 (各域名独立记忆) |
| `+` / `-` | 🔍 **手动缩放** | 调整当前媒体的放大倍率 |
| `R` / `M` | 📐 **旋转 / 镜像** | 顺时针旋转90度 / 水平翻转 |
| `Esc` / 双击 | ❌ **退出沉浸模式** | 恢复正常网页浏览 |

## ⚙️ 控制面板与个性化 (Options)

点击扩展图标进入设置页面：
* **域名记忆隔离**：你在推特设置了“整体跟随”模式，在 Pixiv 设置了“局部放大”模式，系统会自动记忆当前域名的偏好，互不干扰。
* **小型图片智能优化**：对小于 `50px` 或 `100px` 的缩略图（如头像），自动分配 `9x` 或 `6x` 的超高初始放大倍率。
* **CDN 高清解析开关**：自由选择是否消耗带宽加载原图。
* **按键全自定义**：所有上述快捷键均可在面板中自由修改。

## 📦 安装说明 (Installation)

由于本扩展包含强大的 DOM 逆向与跨域下载功能，未上架 Chrome Web Store，请以开发者模式加载：

1. 下载本项目的 `.zip` 源码包并解压到一个文件夹中。
2. 打开 Chrome / Edge 浏览器，在地址栏输入 `chrome://extensions/`。
3. 开启右上角的 **开发者模式 (Developer mode)**。
4. 点击左上角的 **加载已解压的扩展程序 (Load unpacked)**。
5. 选择你刚刚解压的文件夹。
6. *首次使用请点击右上角扩展图标，阅读并同意免责声明。*
